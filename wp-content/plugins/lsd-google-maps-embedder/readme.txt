== Description ==

With this plugin you can easily add different types of Google Maps embeds. The places, search, directions and view maptypes. (See screenshots for differences between the maptypes.)

A Google Maps API key is needed, but can be obtained for free. A step by step explaination is included in the plugin-admin page.

== Installation ==

1. Unzip the archive in the wp-content/plugins folder.
2. Activate via the plugin activation screens.
3. Enter your Google Maps API key in Settings > Google Maps (How to get a key is explained step by tep on that page)
